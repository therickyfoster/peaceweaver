## üÜò Troubleshooting

### Map Not Showing?
- **Check**: Browser console for errors (F12)
- **Fix**: Ensure JavaScript is enabled
- **Fix**: Try refreshing page (Ctrl+R or Cmd+R)

### Animations Stuttering?
- **Check**: Other tabs consuming resources
- **Fix**: Close unnecessary browser tabs
- **Fix**: Disable browser extensions temporarily

### Colors Look Different?
- **Check**: Display color profile settings
- **Check**: Night mode or color filters active
- **Note**: Design optimized for standard sRGB displays

### Calculator Not Working?
- **Check**: All fields have valid numbers
- **Fix**: Ensure duration is between 1-30
- **Fix**: Ensure efficiency is between 50-100
